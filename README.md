
# 📌 Project Title
Stopwatch Web Application

# 📖 Project Overview
The Stopwatch Web Application is a simple and interactive timer built using HTML, CSS, and JavaScript. It allows users to accurately measure elapsed time with features such as Start, Pause, Reset, and Lap Recording. The application provides a clean and responsive user interface, making it easy to use on both desktop and mobile devices.

This project was developed as part of the SkillCraft Technology Internship Program (SCT_WD_2) to demonstrate JavaScript functionality, DOM manipulation, and event handling.

# ✨ Features
▶️ Start the stopwatch

⏸️ Pause the stopwatch

🔄 Reset the stopwatch

# 📱 Fully responsive design

🎨 Modern and user-friendly interface

⚡ Real-time time tracking with milliseconds

# 🛠️ Technologies Used
. HTML5 – Structure of the application

. CSS3 – Styling and responsive design

. JavaScript (ES6) – Stopwatch functionality and lap management
